# Vue-Tracker
全栈轻架构的项目进度跟踪管理工具**Atomic Tracker**的后端部分。
后端架构为NestJS，其中使用MongoDB数据层，Mongoose为ORM层, Graphql为数据聚合层。 
## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
